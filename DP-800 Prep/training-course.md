# Official Training Course: DP-800 (Developing AI-Enabled Database Solutions)

**Course:** Develop AI-enabled database solutions (DP-800T00)
**URL:** https://learn.microsoft.com/en-us/training/courses/dp-800t00
**Certification:** Microsoft Certified: SQL AI Developer Associate (beta)
**Certification URL:** https://learn.microsoft.com/en-us/credentials/certifications/developing-ai-enabled-database-solutions/

**Level:** Intermediate

> **Note:** DP-800 is currently in beta. The official Microsoft Practice Assessment is not yet available (typically available within 8 weeks of GA).

---

## Learning Path 1: Design and Develop Database Solutions
- **Duration:** 7 hr 11 min | 4 Modules
- **URL:** https://learn.microsoft.com/en-us/training/paths/design-develop-database-solutions/
- **Exam Weight:** 35–40%

| Module | Duration | Description |
|--------|----------|-------------|
| Design and implement database objects with SQL | 1 hr 38 min (11 units) | Tables, data types, indexes, columnstore indexes, specialized tables (in-memory, temporal, external, ledger, graph), JSON columns, constraints, sequences, partitioning |
| Implement programmability objects with SQL | 1 hr 30 min (10 units) | Views, stored procedures, scalar functions, table-valued functions, triggers |
| Write advanced T-SQL code | 2 hr 26 min (12 units) | CTEs, window functions, JSON functions, regular expressions, fuzzy string matching, graph queries (MATCH), correlated queries, error handling |
| Implement SQL solutions by using AI-assisted tools | 1 hr 37 min (10 units) | GitHub Copilot, Fabric Copilot, MCP tool configuration, instruction files, connecting to MCP server endpoints |

**Prerequisites:** Experience writing T-SQL queries, basic understanding of database concepts (tables, joins, transactions), familiarity with SQL Server / Azure SQL Database / SQL databases in Fabric.

---

## Learning Path 2: Secure, Optimize, and Deploy Database Solutions
- **Duration:** 7 hr 31 min | 4 Modules
- **URL:** https://learn.microsoft.com/en-us/training/paths/secure-optimize-deploy-database-solutions/
- **Exam Weight:** 35–40%

| Module | Duration | Description |
|--------|----------|-------------|
| Implement data security and compliance with SQL | 2 hr 1 min (11 units) | Always Encrypted, column-level encryption, Dynamic Data Masking, Row-Level Security, object-level permissions, auditing, securing model endpoints, securing GraphQL/REST/MCP endpoints |
| Optimize database performance | 1 hr 44 min (9 units) | Service tier selection, transaction isolation levels, execution plans, DMVs, Query Store, Query Performance Insight, blocking & deadlocks |
| Implement CI/CD by using SQL Database Projects | 1 hr 56 min (10 units) | Testing strategy (unit + integration), source control, branching, pull requests, secrets management, schema drift detection, deployment pipelines, SDK-style models |
| Integrate SQL solutions with Azure services | 1 hr 50 min (10 units) | Data API Builder (DAB), REST & GraphQL config, entity configuration, DAB deployment, Azure Monitor/App Insights/Log Analytics, change event streaming (CES), CDC, Change Tracking, Azure Functions SQL triggers |

**Prerequisites:** Experience writing T-SQL, familiarity with SQL Server or Azure SQL Database, basic security concepts (authentication/authorization), experience with Git and CI/CD concepts.

## Learning Path 3: Implement AI Capabilities in Database Solutions
- **Duration:** 3 hr 40 min | 3 Modules
- **URL:** https://learn.microsoft.com/en-us/training/paths/implement-ai-capabilities-database-solutions/
- **Exam Weight:** 25–30%

| Module | Duration | Description |
|--------|----------|-------------|
| Design and implement models and embeddings with SQL | 1 hr 1 min (8 units) | External model evaluation (multimodal, multilanguage, sizes), creating/managing external models, embedding maintenance methods, column selection for embeddings, chunk design, embedding generation |
| Design and implement intelligent search with SQL | 1 hr 24 min (9 units) | Full-text search, vector data type/indexes, VECTOR_NORMALIZE/VECTOR_DISTANCE/VECTOR_SEARCH functions, ANN vs ENN, vector index types/metrics, hybrid search, reciprocal rank fusion (RRF), performance evaluation |
| Design and implement RAG with SQL | 1 hr 15 min (8 units) | RAG use cases, prompt creation via sp_invoke_external_rest_endpoint, converting structured data to JSON for LLM processing, sending results to language models, extracting LLM responses |

**Prerequisites:** Experience with Azure SQL Database or SQL Server, writing T-SQL queries, general understanding of AI concepts.
